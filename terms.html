Пользовательское соглашение

Настоящее пользовательское соглашение (далее — «Соглашение») регулирует отношения между сервисом «Толкование Снов» и Пользователем, использующим один или несколько разделов: «Сонник», «Гороскоп», «Таро», «Хиромантия» (далее — «Сервис»).

1. Общие положения

  • Используя Сервис, Пользователь соглашается с условиями настоящего Соглашения.
  • Если Пользователь не согласен с условиями, он должен прекратить использование Сервиса.

2. Обработка данных

  • Мы собираем: имя, дату рождения, текст снов, текстовые вопросы, фотографии рук.
  • Данные используются только для генерации ответа с помощью ИИ и не передаются третьим лицам (кроме OpenAI API и n8n).
  • Фотографии не хранятся на сервере, удаляются после анализа.

3. Ограничение ответственности

  • Все результаты имеют развлекательный характер и не являются консультациями.
  • Сервис не несет ответственности за последствия использования предоставленной информации.

4. Права и обязанности

  • Пользователь обязуется не нарушать законодательство и не загружать запрещённые материалы.
  • Сервис может изменять функционал, блокировать доступ при нарушении правил.

5. Заключительные положения

  • Соглашение может быть изменено в любой момент без уведомления.
